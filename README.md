# WDC UDAAN WEBSITE

Modern Women Development Cell UDAAN platform built with **React + Vite + Tailwind CSS**.

## ✨ Features

- Responsive dashboard layout (sidebar + topbar)
- Glassmorphism design with pastel gradients
- Framer Motion animations
- React Router navigation
- Events, Complaints management
- Multi-step forms
- Mock data implementation
- Production-ready UI

## 🚀 Quick Start

```bash
npm install
npm run dev
```

## 📱 Pages

- `/` - Landing
- `/login` - Login
- `/register` - Register
- `/home` - Dashboard  
- `/events` - Events list
- `/events/:id` - Event details
- `/complaints` - Complaints list
- `/complaint/new` - New complaint form

## 🎨 Design

- Pastel purple/pink gradient theme
- Glassmorphism cards
- Smooth animations & transitions
- Fully responsive (mobile-first)

## 📁 Structure

```
src/
├── components/     # Reusable UI components
├── layouts/        # Page layouts
├── pages/          # Page components
├── data/           # Mock data
└── utils/          # Utilities
```
